### Inputs
The project uses live stock market parameters as inputs:
- Symbol, Open, High, Low, Previous Close, Last Traded Price (LTP), Indicative Close
- Daily Change and % Change
- Trading Value (in Crores)
- 52-Week High/Low
- 30-Day % Change and 365-Day % Change

These inputs are processed to predict the **closing price** and **trading volume** for the selected stock.
